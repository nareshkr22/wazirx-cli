# Wazirx-cli
Python CLI program to track your P&amp;L on WazirX platform

## Config File
Add your existing portfolio details in the config file in the following format
cryptoname qty price 
```
dogeinr	92	26.38
xrpinr	17	111.49
wrxinr	7	260.71
usdtinr	20.35	83.01
```
## Prerequisite
```
pip install texttable
pip install requests
```

## Run
``` 
python wazirx.py
```
